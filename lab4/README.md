# Лабораторная работа 4: Множественная линейная регрессия

## Описание

Построение множественной линейной регрессионной модели для анализа зависимости численности населения от экономических и экологических факторов.

## Задачи

1. Провести корреляционный анализ и отобрать значимые факторы
2. Построить множественную линейную регрессию (МЛР)
3. Оценить значимость коэффициентов регрессии (t-критерий Стьюдента)
4. Проверить адекватность модели (F-критерий Фишера, R²)
5. Выполнить анализ остатков и проверить предпосылки МНК
6. Интерпретировать результаты

## Модель

### Зависимая переменная

- **y** - численность населения (млн человек)

### Независимые переменные (факторы)

- **x1** - ВВП (млрд долларов США)
- **x2** - уровень выброса CO₂ (тонн на душу населения)
- **x3** - энергопотребление (кВт·ч на душу населения)
- **x4** - расходы на здравоохранение (% от ВВП)
- **x5** - уровень урбанизации (%)

## Используемые методы

### 1. Отбор факторов

- Корреляционный анализ (коэффициент Пирсона)
- VIF-анализ (проверка мультиколлинеарности)

### 2. Построение модели

- Метод наименьших квадратов (МНК)
- OLS-регрессия через statsmodels

### 3. Оценка значимости коэффициентов

- t-критерий Стьюдента
- p-value анализ
- Доверительные интервалы (95%)

### 4. Оценка адекватности модели

- F-критерий Фишера (общая значимость)
- Коэффициент детерминации R²
- Скорректированный R²
- RMSE, MAE, MAPE

### 5. Проверка предпосылок МНК

- Нормальность остатков (тест Шапиро-Уилка, Харке-Бера)
- Автокорреляция остатков (тест Дарбина-Уотсона)
- Гомоскедастичность (визуальный анализ)
- Анализ выбросов

## Интерпретация результатов

### Коэффициент детерминации (R²)

- R² ≥ 0.8: отличное качество модели
- 0.6 ≤ R² < 0.8: хорошее качество
- 0.4 ≤ R² < 0.6: удовлетворительное качество
- R² < 0.4: низкое качество

### Значимость коэффициентов (p-value)

- p < 0.05: коэффициент статистически значим
- p ≥ 0.05: коэффициент незначим (можно исключить)

### Адекватность модели (F-test)

- p-value < 0.05: модель адекватна
- p-value ≥ 0.05: модель неадекватна

### VIF (проверка мультиколлинеарности)

- VIF < 5: отсутствие мультиколлинеарности
- 5 ≤ VIF < 10: умеренная мультиколлинеарность
- VIF ≥ 10: высокая мультиколлинеарность (исключить фактор)

### Тест Дарбина-Уотсона (автокорреляция)

- 1.5 ≤ DW ≤ 2.5: автокорреляция отсутствует
- DW < 1.5: положительная автокорреляция
- DW > 2.5: отрицательная автокорреляция
